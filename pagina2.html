Curso Básico de HTML




Una vez creado el documento y habiendo establecido la estructura básica, se podrá escribir dentro del body cualquier texto que se desee. Lo importante es, mediante estas etiquetas que te damos a continuación, darle formato al texto. Para comprobar que las entiendes perfectamente, podrás realizar un ejercicio directamente en la página y comprobar la solución si tienes algún tipo de problema:


Etiquetas

Efecto de la etiqueta

<br> Espacio entre letras
<strong> </strong> Negrita
<i> </i> o <em> </em> Cursiva
<u> </u> Subrayado
<mark> </mark> Resaltado
<s> </s> o <del> </del> Texto tachado
<big> </big> Letra un poco más grande
<small> </small> Letra un poco mas pequeña
<!__comentario__> Comentario para el programador (no se ve a la pagina WEB)
<center> </center> Menú Estudiante
Prueba a escribir las etiquetas de la izquierda en este recuadro de texto para comprobar el resultado. Un ejemplo sería:
<strong> Este texto está en negrita </strong>
<br> {Aquí habría dos saltos de línea} </br> <i> Despues se escribiría en la tercera linea en cursiva. </i>
Prueba a copiar y pegar en el recuadro este breve código y presiona el botón convertir.





Además podrás Cambiar el tipo de letra, tamaño y color, mediante la siguiente etiqueta:

<font face ="TipoDeLetra" size="TamañoDeLetra" color="ColorDeLetra"> TextoAfectado </font>

Ejemplo:
<font face ="Arial" size="11" color="Red"> Este es el texto que tendrá las propiedad designadas. </font>


Prueba a copiar y pegar también este codigo en el
cuadro para que puedas comprobarlos resultados.
